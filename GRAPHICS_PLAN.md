# 두더지 게임 그래픽 계획서 (이미지 기반)

## 1. 기준 이미지 분석
- 참조 파일:
1. `image.png` (1536x1024)
2. `image copy.png` (1536x1024)

- 스타일 요약:
1. 밝은 캐주얼 카툰
2. 두꺼운 외곽선 + 고채도 색상
3. 잔디/흙/나무/울타리 중심의 야외 테마
4. 친근한 캐릭터 표정(아동/캐주얼 모바일 톤)

- 결론:
1. `title/브랜딩`은 `image.png`가 강점
2. `실제 플레이(3x3~9x9 가변)`는 구멍이 고정 배치된 배경보다, 배경과 오브젝트를 분리해야 확장성이 좋음

## 2. 그래픽 전략 (실전형)

### 핵심 원칙
1. 배경은 정적(1~2장)으로 유지해 성능 안정화
2. 구멍/두더지/망치/VFX는 스프라이트로 분리
3. 보드 크기(3x3~9x9)는 코드로 동적 배치

### 기존 이미지 사용 방안
1. `image.png`:
   - 타이틀 화면 Key Visual
   - 튜토리얼/로딩 배경
2. `image copy.png`:
   - 플레이 배경의 아트 기준 레퍼런스
   - 실제 게임에서는 "빈 잔디 배경판"으로 재제작 권장

## 3. 레이어 구조

### 플레이 씬 레이어 (Canvas)
1. Layer 0: 하늘/나무/울타리/원경 배경
2. Layer 1: 잔디/흙 바닥(중경)
3. Layer 2: 구멍(동적 배치, 3x3~9x9)
4. Layer 3: 두더지 스프라이트(타입별)
5. Layer 4: 타격 이펙트(별/스파크/흙 파편)
6. Layer 5: 콤보 게이지/숫자 팝업(오버레이)

### UI 레이어 (HTML 오버레이)
1. 상단 HUD: 점수/시간/레벨
2. 우측 하단: 일시정지/설정
3. 결과 모달: 점수, 보상 버튼, 광고 진입 버튼

## 4. 3x3~9x9 대응 배치 규칙

### 보드 안전 영역
- 기준 캔버스(가로형): 1536x1024
- 권장 보드 영역:
1. 좌우 여백 포함 `x: 12% ~ 88%`
2. 상하 여백 포함 `y: 24% ~ 86%`

### 셀 배치
1. `N x N` 격자에서 `N=3..9`
2. 셀 중심 좌표는 균등 분할
3. 원근 느낌은 `row`가 아래로 갈수록 구멍 크기 +6~10% 확장
4. 히트박스는 타원형(가로 1.0, 세로 0.72 비율) 사용

### 권장 크기 가이드
1. 3x3: 구멍 직경 120~150px
2. 6x6: 구멍 직경 84~108px
3. 9x9: 구멍 직경 58~76px

## 5. 스프라이트 사양

### 구멍
1. 상태: idle, hit(먼지), cooldown
2. 권장: 256x256 PNG 시트, 3~5프레임

### 두더지 타입
1. 일반: 등장/대기/피격/퇴장
2. 황금: 반짝임 루프 + 피격
3. 폭탄: 경고 점멸 + 폭발
4. 얼음: 냉기 파티클 + 슬로우 발동

### 망치/이펙트
1. 망치 스윙 4~6프레임
2. 타격 스파크 6~8프레임
3. 점수 팝업(+1,+3,-3) 비트맵 폰트 또는 SDF 폰트

## 6. 색/톤 가이드

### 팔레트 방향
1. 메인 그린: 잔디/자연톤
2. 보조 브라운: 흙/구멍/나무
3. 강조 옐로우/레드: 점수, 콤보, 경고
4. UI 블루: 점수판/버튼 기본

### 시인성 규칙
1. 인터랙션 요소는 배경 대비 4.5:1 이상
2. 폭탄/패널티는 빨강 계열 고정
3. 황금 보상은 노랑 + 외곽 글로우

## 7. 애니메이션 계획

### 등장/퇴장 타이밍
1. 등장: 120~180ms ease-out
2. 대기: 레벨별 700~1200ms
3. 퇴장: 100~140ms ease-in

### 타격 피드백
1. 히트 스탑: 35~55ms
2. 카메라(화면) 셰이크: 2~4px, 60ms 이내
3. 콤보 임계치(5/10/15)에서 글로우 강화

## 8. 제작 우선순위
1. 배경 1장(플레이 공용)
2. 구멍 스프라이트 1종
3. 두더지 4타입(일반/황금/폭탄/얼음)
4. 타격 이펙트 2종(일반/크리티컬)
5. HUD 버튼/패널 세트
6. 결과/상점 모달 스킨

## 10. AI 이미지 생성 요청 문구 예시
아래 문구를 그대로 이미지 생성에 사용 가능하다.

### 배경
`밝은 캐주얼 카툰 스타일의 두더지 게임 배경, 하늘과 울타리, 잔디와 흙길, 중앙은 오브젝트 배치를 위한 넓은 빈 공간, 구멍 없음, 고채도, 두꺼운 외곽선, 모바일 게임 UI 친화적, 1536x1024, transparent 요소 없음`

### 구멍
`카툰 스타일의 흙 구멍 오브젝트 1개, 위에서 약간 내려다본 시점, 두꺼운 라인, 명확한 안쪽 그림자, 투명 배경 PNG, 512x512`

### 두더지(일반)
`귀여운 카툰 두더지 캐릭터 스프라이트 시트, 등장/대기/피격/퇴장 4가지 포즈, 통일된 조명, 투명 배경, 모바일 게임용, 1024x1024`

## 11. 구현 연결 규칙
1. 캔버스에는 배경 1장 + 반복 가능한 구멍/캐릭터 스프라이트만 사용
2. 점수/시간/버튼은 HTML 오버레이 유지(광고/반응형 대응)
3. 모든 리소스는 WebP 우선, 투명 필요 자산만 PNG 유지
4. 저사양 대응으로 1x/2x 해상도 세트 제공
